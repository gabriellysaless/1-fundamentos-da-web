# 🌐 Fundamentos da Web

> **Repositório destinado aos estudos e exercícios desenvolvidos durante o módulo de Fundamentos da Web da EBAC.**

Este repositório reúne os projetos e exercícios realizados ao longo do módulo, abordando os principais conceitos para o desenvolvimento de páginas web modernas.

O objetivo é documentar minha evolução no aprendizado, praticar boas práticas de desenvolvimento e servir como material de consulta durante minha formação em Desenvolvimento Front-end.

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript

## 📚 Conteúdo estudado

- Estrutura básica de uma página HTML
- Tags semânticas
- Listas
- Tabelas
- Formulários
- Links e navegação
- Imagens
- Áudio e vídeo
- Introdução ao CSS
- Seletores
- Cores
- Tipografia
- Box Model
- Flexbox
- Posicionamento de elementos

## 📂 Estrutura do repositório

```text

```

> A organização das pastas acompanha a sequência dos módulos e exercícios desenvolvidos durante o curso.

## ⚙️ Como executar

Como a maioria dos projetos utiliza apenas HTML e CSS, basta abrir o arquivo `index.html` no navegador.

Caso utilize o VS Code, recomenda-se a extensão **Live Server** para facilitar o desenvolvimento.
